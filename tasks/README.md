# web-services-and-applications
Assignments for the course "Web services and applications", Higher Diploma in Computing for Data Analytics, ATU Galway Mayo, 2026. 

## In this folder: 

1. Card draw: a program to deal cards and contratulate the user on a good hand. 

2. CSO: retrieve data from the CSO and dump it in a .json file.

3. GitHub: use the gihub API to access a repo, modify content and push to the repo. 

## Other 
It's not required to install additional dependencies. All the modules used in these programs are built-in python modules.
