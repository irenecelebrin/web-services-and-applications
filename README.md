# web-services-and-applications
Work, assignments and final project for the course "Web services and applications", Higher Diploma in Computing for Data Analytics, ATU Galway Mayo, 2026. 

## In this repository: 

1. **Notes and Labs**: individual work during the course. 

2. **Tasks**: weekly assignments for the course. 

3. **Project**: final project for the course. The project is a Rest API that allows admins of the association Slackline Ireland to manage members, emergency contacts and payments.
